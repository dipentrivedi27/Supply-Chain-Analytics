# 🚚 Supply Chain Analytics Dashboard

## 📌 Project Overview

This project focuses on analyzing supply chain operations using Python, MySQL, and Power BI. The goal was to transform raw supply chain data into actionable business insights through data cleaning, SQL analysis, and interactive dashboards.

The dashboard helps stakeholders monitor sales performance, profitability, customer behavior, market trends, logistics efficiency, and order management.

## 🧹 Data Preparation

### Data Cleaning

Performed the following preprocessing steps:

* Removed unnecessary columns

  * Customer Email
  * Customer Password
  * Customer First Name
  * Customer Last Name
  * Customer Street
  * Product Image
  * Latitude
  * Longitude

* Checked for missing values

* Verified data types

* Removed irrelevant fields for business analysis
